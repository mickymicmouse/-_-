### 키의 개념 및 종류

* 슈퍼키
  * 유일성은 만족하지만 최소성 만족못함

  * 슈퍼키의 부분 집합
  * 유일성과 최소성 만족
* 기본키
  * 후보키 중에서 선정된 키로 중복값을 가질 수 없는 것
  * Null 값 X
* 대체키

* 복합키

### 무결성

* 개체 무결성
  * 기본키를 구성하는 어떤 속성도 NULL이나 중복값을 가질 수 없다
* 도메인 무결성
  * 정의된 도메인에 속한 값이어야 한다.
* 참조 무결성
  * 외래키는 참조 릴레이션의 기본키 값과 동일해야함
* 사용자 정의 무결성
  * 속성 값들이 사용자 정의에 맞아야 함
* NULL 무결성
  * 특정 속성 값이 NULL이 될 수 없게 하는 것
* 고유 무결성
  * 특정 속성이 중복이 있어서는 안된다.
* 키 무결성
  * 하나의 릴레이션에는 적어도 하나의 키가 존재
* 관계 무결성
  * 릴레이션에 한 튜플의 삽입, 다른 릴레이션들의 튜플들 사이의 관계의 적절성 여부

### 네트워크 관련 장비

* 네트워크 인터페이스 카드
  * 이더넷카드, 네트워크 어댑터라고 함
  * 컴퓨터와 네트워크 연결
* 허브
  * 가까운 거리의 컴퓨터들을 연결
  * 신호 증폭 기능의 리피터의 역할 포함
  * 더미 허브
    * 단순 연결(물리적인 성형)
  * 스위칭 허브
    * 최대 대역폭을 사용할 수 있는 지능형 허브
* 리피터
  * 오류 발생시 전송되는 신호 재전송
  * 신호 증폭 역할
* 브리지
  * LAN 과 LAN을 연결
  * 전송가능 회선수 = n*(n-1)/2
* 스위치
  * LAN + LAN = 더큰 LAN
* 라우터
  * 브리지처럼 연결 기능 + 데이터 전송의 최적 경로 탐색
* 게이트 웨이
  * 전 계층의 연결
